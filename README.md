##### Abdurrahman Jahankeer - Assignment Tracker (ExpressJS, MongoDB)
Personal Assignment Tracker web app built with Node.js, Express.js, EJS, and MongoDB for INFR3120 “Web and Script Programming” at Ontario Tech University.

## Live Site
https://abdurrahman-jahankeer-infr3120-fall25.onrender.com/ 

## Project Plan
https://docs.google.com/document/d/1qXjKDMwKGQTozT8--BLIYI6NKkeXARnXiIP3T7TqyaI/edit?usp=sharing 

## Features
Home Page
About Me Page (with picture and video)
Assignments Page (Contains 5 assignment pages so far):
- Lists all assignments (Read): Table view of every assignment in the tracker.
- Add Assignment (Create): Button/form to add a new assignment.
- Edit Assignment (Update): Pencil/edit button opens a form to update assignment info.
- Delete Assignment (Delete): Trash button lets you remove an assignment (with confirmation).
Contact Page

## Buttons/Actions on Assignments Page
- Add Assignment to the assignment list
- Edit Assignment from the assignment list
- Delete an Assignment from the assignment list

## Installation and Website Starting Instructions
- Download the Github Repository file
- Extract the file
- Open the folder on Visual Studio Code
- Open new terminal
- Then type "npm i -g nodemon"
- Then type "nodemon" to run the server or EJS website
- Lastly, go on a browser and open either "http://localhost:3000" or "127.0.0.1:3000" in the URL search bar
